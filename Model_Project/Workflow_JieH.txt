Say the home directory in Derecho is /glade/u/home/$User
And the CTSM (ctsm5.3.069) codes are stored in $CTSMROOT
And the model simulation directory is $CASEROOT
Test site is $Site

The order of workflow:
0. Upload the FLUXNET forcing data to my Derecho
	scp -r C:/Users/.../1x1pt_DE-Tha $User@casper.hpc.ucar.edu:/$CASEROOT/inputdata/atm/datm7/CLM1PT_data

1. Use the crop tool '$CTSMROOT/tools/site_and_regional/subset_data' to create the CLM5 surface files for the test site

2. Create a new AD spinup case "${Site}_FLUX_AD" using I2000Clm50BgcCruRs compset
	go to '$CTSMROOT/cime/scripts/'
	create_newcase

3. Make ./xml changes and edit user_nl_clm and user_nl_datm_streams
  following DE-Tha_AD_FLUXNET.txt

4. Submit the ${Site}_FLUX_AD job to Derecho to run the AD spin-up
	Took ~3hrs for 200 years of AD spin-up

5. Create a new regular spinup case "${Site}_FLUX_FS" using the same I2000Clm50BgcCruRs compset

6. Make ./xml changes and edit user_nl_clm and user_nl_datm_streams
  following DE-Tha_FN_FLUXNET.txt

7. Submit the ${Site}_FLUX_FS job to Derecho to run the final spin-up
	Took ~2hrs for 100 years of regular spin-up

8. Create a new transient case "${Site}_FLUX_Transient" using the same compset

9. Make ./xml changes and edit user_nl_clm and user_nl_datm_streams
  following DE-Tha_Transient_FLUXNET.txt
  
10. Submit the ${Site}_FLUX_Transient job to Derecho to run the transient period

11. Download the output histfiles to your local computer
   For example,	
   scp -r ${User}@casper.hpc.ucar.edu:/glade/derecho/scratch/${User}/archive/$Site_FLUX_Transient/lnd/hist C:/Users/.../${Site}_FLUX_Transient
   
12. Concatenate daily histfiles to one nc file
	Load the nco module first
	for f in *.nc; do echo ${f%.h1a*}; done | sort | uniq | while read prefix; do ncrcat "$prefix".h1a.*.nc "$prefix".nc; done
